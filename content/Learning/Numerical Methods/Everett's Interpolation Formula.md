# Everett's Interpolation Formula
- Use the [[Central Difference]] Table
- $s = 1-r$

> [!tldr]  Formula:
> $$
> f_x = f(x_0 + rh) = 
> sf_0 + \frac{(s)(s^2-1)}{3!} \delta^2 f_0 +
> \frac{(s)(s^2-1)(s^2-4)}{5!} \delta^4 f_0 +
> ... +
> rf_1 + \frac{(r)(r^2-1)}{3!} \delta^2 f_1 + 
> \frac{(r)(r^2-1)(r^2-4)}{5!} \delta^4 f_1
> $$
>
