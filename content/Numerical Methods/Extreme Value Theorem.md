# Extreme Value Theorem
- Gets the maximum and minimum values for a function
- Just get $x$ from $f'(x)$

## Example
$f(x) = \frac{1}{6}(x^3-6x^2+9x+6$
$f'(x) = \frac{1}{6}(3x^2-12x+9) = \frac{1}{2}(x^2-4x+3)$
$f'(c) = \frac{1}{2}(c^2-4c+3) = 0$
$f'(c) = c^2 -4c +3 = 0$
$(x-3)(x-1) = 0$
$x_1 = 3; x_2 = 1$

Minimum and Maximum Values:
$f(3) = 1$
$f(1) = \frac{5}{3}$
